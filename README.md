# polls_django
using django to create a web-application to create a poll.

## RUN

### Requirements

Django

`pip3 install django`

Clone the repository or Download as zip

Enter the Repo

`cd polls_django`

Run manage.py

`python3 manage.py runserver`

open any browser and enter the following URL

`http://127.0.0.1:8000/polls`
